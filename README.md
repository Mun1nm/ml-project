## Como Executar
1. Clone o repositório e entre na pasta do projeto.
2. Crie e ative um ambiente virtual.
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
4. Abra o notebook:
   ```bash
   jupyter notebook main.ipynb
   ```
